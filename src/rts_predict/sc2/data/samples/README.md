# SC2EGSet: StarCraft II Esport Game State Dataset

This repository contains pre-processed data that was generated from the [SC2ReSet: StarCraft II Esports Replaypack Set](https://doi.org/10.5281/zenodo.5575796)

## Data Modeling

Our official aplication programing interface (API) implementation supports downloading, unpacking, loading and data access features. Please refer to: [GitHub SC2_Datasets](https://github.com/Kaszanas/SC2_Datasets)

## Changelog

Please refer to the CHANGELOG.md file within this repository for full changelog.
